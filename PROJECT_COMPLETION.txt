HOW TO USE THE PROJECT:
-----------------------

1. Clone from GitHub:
   git clone https://github.com/[USERNAME]/air-quality-blockchain.git

2. Install dependencies:
   npm install

3. Compile contract:
   npx hardhat compile

4. Run tests:
   npx hardhat test

5. Measure gas:
   npx hardhat run scripts/measureGas.js --network hardhat

6. Deploy:
   npx hardhat run scripts/deploy.js --network hardhat
