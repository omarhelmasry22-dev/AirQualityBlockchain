# Smart City Air Quality Blockchain

Secure authentication and trusted data recording for IoT air-quality sensors using Ethereum smart contracts.

## Project Overview

This project implements a smart contract for recording air quality data on the blockchain with:
- Sensor authentication
- Immutable data storage
- Replay attack protection
- Role-based access control
